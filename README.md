# 🐾 Puurfect Pals – Pet Adoption Platform

---

## 📖 Project Overview

**Puurfect Pals** is a full-stack web application designed to simplify the pet adoption and pet care process. The platform provides a centralized system where users can browse pets, add their own pets for adoption, adopt available pets, and book appointments with veterinary doctors. Administrators manage pets, doctors, and platform content to ensure a smooth and organized adoption experience.

### 🔹 Project Summary

- 🐾 Digital platform for pet adoption and pet care
- 👥 Supports two user roles: User and Administrator
- 🏠 Allows users to add pets for adoption
- ❤️ Enables users to adopt available pets
- 👨‍⚕️ Supports online veterinary doctor appointments
- 📷 Pet image upload support
- 🔐 Secure user authentication
- 📊 Centralized pet and doctor management
- ✅ Simplifies the pet adoption and healthcare process

---

## ✨ Key Features

### 👤 User

- Register and log in securely
- Browse available pets
- View detailed pet information
- Add pets for adoption
- Edit personal profile
- Submit pet adoption requests
- Book veterinary doctor appointments
- View appointment details
- Manage personal pets

### 👨‍💼 Administrator

- Log in to the admin panel
- Manage registered users
- Add new pets
- Update pet information
- Delete pet records
- Add veterinary doctors
- Update doctor information
- Delete doctor records
- Monitor adoption requests
- Access dashboard overview
